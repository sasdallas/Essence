# Essence

Essence is a silly bash-like shell I wrote for Ethereal. It runs on Linux too though.

## What is supported?

Statements like `if`, `while` are supported. Most Bash-like syntax is supported, such as `$(cmd)` and others.

## How do I build it?

Run `make`
